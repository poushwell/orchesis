## Неточности по заданию (не баг)

### Неточность #1: в документации есть `--config`, а в CLI такого параметра нет
- **Что заметил:** в тексте задания/документации встречается запуск с `--config`, но команда `orchesis` этот параметр не понимает.
- **Как сейчас работает:** вместо этого используется `--policy` (то есть один файл с настройками и правилами).
- **Почему это важно:** человек читает документацию, пробует команду “как написано” и получает ошибку, хотя сам продукт работает.
- **Вывод:** это не поломка кода, а расхождение между формулировкой в задании/доках и реальным интерфейсом CLI.
